Swedroid Communitys Ikonpaket - Novum Icons

Initiativet grundar sig på att inga ikonpaket är kompletta, och att det alltid saknas en eller flera ikoner. Ingen PSD finns tillgänglig så man kan inte komplettera med de ikoner man själv saknar. 

Lösningen är naturligtvis att skapa ett ikonprojekt där det finns en PSD för projektet, där många bidrar, och där man snabbt kan skapa de ikoner som man saknar. Allt samlas upp i en projekt-mapp där alla ikoner som är i linje med projektets ramar och design hamnar!

Idén är enkel. Vi ska skapa sjukt mycket ikoner, och förhoppningsvis kan det bli ett riktigt stort ikonpaket som kan utmana de stora drakarna!
